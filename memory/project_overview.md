libs3 is a pure C23 single-header-file library for interacting with Amazon S3, built on top of libcurl.

**Why:** Provide a lightweight, easy-to-integrate S3 client with no dependencies beyond libcurl.

**How to apply:** All code goes in a single header file. Target C23 standard features. Depend only on libcurl for HTTP.
